Overview

Android project developed to help color blind people see what color they are currently looking at by taking a picture. Then using RGB color thresholding to determine against a database to see what color it is, displaying it in color neutral white on black with a name and a hex number.

Features
  • Analyzing within 0.1 second what color it is.
  • Supports clicking within one picture to determine what color anything in the picture is.
  • 2 modes, basic and detailed with basic mode showing the colors in the rainbow while detailed mode has 500+ color choices including colors like "pearl" or "oak".
Languages Used

Java in Android Studio
